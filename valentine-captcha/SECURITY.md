# Security Policy

## Use of this Code

This project is open for everyone. You are free to use, modify, and distribute this code for any purpose.

## Supported Versions

We support all versions of this project with security updates.

| Version | Supported          |
| ------- | ------------------ |
| All     | :white_check_mark: |

## Reporting a Vulnerability

To report a vulnerability, please open an issue in this repository. We aim to acknowledge reports within 48 hours.
